# responsive-Calculator
A clean, modern, and fully responsive calculator built with HTML5, CSS3, and Vanilla JavaScript. This project provides a seamless user experience across desktop and mobile devices.

🚀 Features
•	Responsive Design: Adapts fluidly to different screen sizes using CSS Grid and Flexbox.
•	Core Arithmetic: Supports basic operations including addition, subtraction, multiplication, and division.
•	User-Friendly Controls: Features an "AC" (All Clear) function and a specialized button layout for improved usability.
•	Visual Feedback: The display area provides immediate visual cues, including error handling with a red glow effect when an invalid calculation is attempted.
•	Minimalist UI: Clean aesthetic with distinct styling for functional buttons.

🛠️ Built With
•	HTML5: Structures the calculator interface.
•	CSS3: Handles styling and the responsive grid layout.
•	JavaScript: Manages calculation logic, string evaluation, and event handling.

📂 Project Structure
├── index.html       # Calculator UI structure
├── style.css        # Responsive styling and layout
└── script.js        # Logic for calculations and display management
